
# libcuishark

libwireshark-suite wrapper lib for cuishark.
easy to test and extend library for packet-analysis super power.

## Install libcuishark

```
$ git clone http://github.com/cuishark/libcuishark
$ cd libcuishark
$ ./bootstrap.sh
$ make && sudo make install
```
